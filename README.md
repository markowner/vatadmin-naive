# vatadmin-naive
基于naiveui开发的管理后台前端页面

### 项目运行
```bash
# 克隆项目
git clone https://github.com/markowner/vatadmin-naive.git

# 进入项目目录
cd vatadmin-naive

# yarn安装依赖
yarn install

# 运行项目
yarn dev
```

### 项目打包
```bash
yarn build
```
